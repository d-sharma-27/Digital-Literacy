## Project Overview
This project was completed as part of the Digital Literacy course. As a Student Digital Ambassador, the objective was to create awareness about digital literacy and develop practical skills for navigating the digital world responsibly. The project covers various aspects such as online safety, professional presence, digital communication, and the use of modern tools for learning and collaboration.

## Module Summaries

Task 1 – Digital Literacy Infographic: Created a one-page infographic using Canva to explain digital literacy, useful tools, safe internet practices, and professional online behavior.

Task 2 – Digital Portfolio: Developed a professional online presence by creating profiles on GitHub, LinkedIn, and Kaggle, and added essential details to each platform.

Task 3 – Platforms Exploration: Solved a beginner-level coding problem on HackerRank and created a Digital Literacy Awareness Quiz using Google Forms.

Task 4 – Email Etiquette: Drafted professional emails and created a checklist for responsible social media usage.

Task 5 – Cybercrime Awareness: Prepared a case study on UPI fraud and created a prevention checklist to promote safe online practices.
